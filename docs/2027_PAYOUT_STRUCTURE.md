# 2027 FLI Golf Season Payout Structure

## Season Overview

**Total Prize Pool**: $4,000,000  
**Number of Tournaments**: 6  
**Distribution Model**: Progressive (later tournaments worth more)  
**Gender Equality**: Equal payouts for men's and women's divisions  
**Franchise Model**: 20% to franchises, 80% to pros

## Payout Distribution Flow

```
Tournament Prize Pool ($500,000 - $833,333)
    │
    ├─► Franchise Cut (20%)
    │   └─► Distributed to franchises based on pro performance
    │
    └─► Pro Cut (80%)
        ├─► Men's Division (50% of pro cut)
        │   └─► Top 20 placements (top-heavy distribution)
        │
        └─► Women's Division (50% of pro cut)
            └─► Top 20 placements (top-heavy distribution)
```

## Gender Equality

**Equal Payouts**: Men and women receive identical amounts for identical placements.

**Example - 1st Place Comparison:**
- Tournament 1: Men's 1st = Women's 1st = $60,000 (pro) + $15,000 (franchise)
- Tournament 6: Men's 1st = Women's 1st = $100,000 (pro) + $25,000 (franchise)

**Equal Opportunities**:
- Same number of paid placements (top 20)
- Same percentage distribution
- Same franchise opportunities

---

## Franchise Model

### How It Works

1. **Pro Competes**: Pro represents their franchise in tournament
2. **Placement Determined**: Pro finishes in paid position (1-20)
3. **Earnings Calculated**: 
   - Pro receives 80% of placement payout
   - Franchise receives 20% of placement payout
4. **Franchise Paid First**: Franchises receive their cut before pros
5. **Pro Paid**: Remaining amount distributed to pro

### Example Scenario

**Tournament**: Championship Finals  
**Pro**: Sarah Johnson (Franchise: Austin Aces)  
**Division**: Women's  
**Placement**: 1st  

**Earnings Breakdown**:
- Total Placement Value: $125,000
- Franchise Cut (20%): $25,000 → Austin Aces
- Pro Cut (80%): $100,000 → Sarah Johnson

**If Austin Aces has 3 pros in top 20**:
- Pro A (1st): $25,000 franchise cut
- Pro B (5th): $4,217 franchise cut
- Pro C (12th): $1,500 franchise cut
- **Total Franchise Earnings**: $30,717
